# ThoughtBloom 실행 가이드 (howToRun)

이 프로젝트는 Vite와 React를 기반으로 구축되었습니다. 단순히 HTML 파일을 더블 클릭하는 방식으로는 작동하지 않으며, 아래의 명령어들을 통해 실행해야 합니다.

## 1. 개발 서버 실행 (추천)
코드를 수정하면서 실시간으로 결과를 확인하고 싶을 때 사용합니다.
```bash
npm run dev
```
- 실행 후 터미널에 나타나는 주소(예: `http://localhost:5173`)를 브라우저에 입력하세요.

## 2. 빌드 결과물 미리보기
실제 배포될 상태의 결과물을 로컬에서 미리 확인하고 싶을 때 사용합니다.
```bash
npm run build
npm run preview
```
- 빌드가 완료된 후 `preview` 명령어를 실행하면 나타나는 주소로 접속하세요.

## 3. GitHub Pages 배포
현재 수정한 내용을 온라인(GitHub Pages)에 반영하고 싶을 때 사용합니다.
```bash
npm run deploy
```
- 이 명령어를 실행하면 자동으로 빌드와 업로드가 진행됩니다. 약 1~2분 뒤에 온라인 주소에서 확인하실 수 있습니다.

---

### ⚠️ 주의사항
*   **하얀 화면 문제**: `dist/index.html` 파일을 브라우저로 직접 열면 보안 및 절대 경로 문제로 인해 화면이 나오지 않습니다. 반드시 위의 명령어들을 사용해 주세요.
*   **환경 설정**: Node.js가 설치되어 있어야 하며, 처음 실행 시 `npm install`을 통해 필요한 패키지들을 설치해야 합니다. (이미 제가 설정을 마쳤으므로 바로 실행 가능합니다.)
