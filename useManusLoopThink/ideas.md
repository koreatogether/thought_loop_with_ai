1. 인용 작업 위임
2. 새로운 주제에 대한 빠른 파악

## 디자인 1: "학문적 우아함" (Academic Elegance)

### 설계 철학
학문적 깊이와 현대적 미니멀리즘을 결합한 디자인. 대학 도서관과 연구실의 침착함을 디지털 환경에 옮겨온 스타일입니다.

### 핵심 원칙
- **정보 위계의 명확성**: 제목, 부제, 본문이 명확하게 구분되어 독자가 쉽게 탐색
- **여백의 철학**: 충분한 여백으로 콘텐츠가 호흡할 수 있는 공간 제공
- **단색 기반 팔레트**: 검정, 회색, 흰색에 깊은 남색 악센트로 신뢰성 강화
- **세리프 폰트의 활용**: 본문에 Georgia나 Merriweather 같은 고전적 세리프 폰트 사용

### 색상 철학
- **배경**: 크림색(#FFFBF0) - 종이의 따뜻함
- **주 텍스트**: 진한 회색(#2C2C2C) - 인쇄된 책의 느낌
- **악센트**: 깊은 남색(#1A3A52) - 학문적 신뢰성
- **강조**: 골드(#D4AF37) - 중요한 개념 표시
- **감정적 의도**: 차분함, 신뢰성, 지식의 깊이

### 레이아웃 패러다임
- 좌측 사이드바(네비게이션)와 우측 메인 콘텐츠 영역
- 각 전략은 카드 형식이 아닌 **문단 기반 에세이 형식**
- 수직 스크롤 중심의 선형 내러티브
- 인용문은 좌측 테두리로 강조

### 서명 요소
1. **수직 선 강조**: 중요한 섹션 좌측에 두꺼운 수직선
2. **마진 노트**: 우측 여백에 작은 보조 설명이나 팁
3. **타이포그래피 변화**: 제목은 대문자, 본문은 소문자로 대비

### 상호작용 철학
- 마우스 호버 시 부드러운 색상 변화
- 클릭 시 페이드 인/아웃 효과
- 스크롤 시 진행 표시기(Progress Bar) 상단에 표시

### 애니메이션
- 페이지 로드 시 텍스트가 위에서 아래로 천천히 나타나기
- 섹션 변경 시 0.3초 페이드 트랜지션
- 호버 시 텍스트 색상이 악센트 색상으로 변경 (0.2초)

### 타이포그래피 시스템
- **제목**: Georgia Bold, 2.5rem, 자간 -0.5px
- **부제목**: Georgia Regular, 1.5rem, 자간 0px
- **본문**: Merriweather Regular, 1.1rem, 줄높이 1.8
- **강조 텍스트**: Georgia Italic

## 디자인 2: "인터랙티브 워크숍" (Interactive Workshop)

### 설계 철학
학습자가 직접 참여하고 실험할 수 있는 워크숍 환경. 각 전략을 **실시간으로 체험**하고 자신의 학습 여정을 기록하는 플랫폼입니다.

### 핵심 원칙
- **참여 중심**: 읽기만 하는 것이 아니라 직접 입력하고 실험
- **시각적 피드백**: 사용자의 행동에 즉각적인 시각적 반응
- **진행 추적**: 사용자가 어느 전략까지 완료했는지 시각적으로 표시
- **커뮤니티 느낌**: 다른 학습자들의 경험을 공유하는 공간

### 색상 철학
- **배경**: 밝은 라벤더(#F5F3FF) - 창의적이고 평온한 분위기
- **주 텍스트**: 진한 보라(#4A3F8F) - 지적 활동성
- **악센트 1**: 산호색(#FF6B6B) - 에너지와 참여
- **악센트 2**: 민트색(#4ECDC4) - 성공과 진행
- **감정적 의도**: 창의성, 참여, 긍정성

### 레이아웃 패러다임
- **카드 기반 그리드**: 각 전략이 상호작용 가능한 카드로 표시
- **진행 바**: 상단에 11개 전략 중 완료한 개수를 시각화
- **사이드 패널**: 현재 전략의 상세 설명과 입력 필드
- **비선형 네비게이션**: 사용자가 원하는 순서대로 전략 탐색 가능

### 서명 요소
1. **진행 원형 게이지**: 각 전략 카드에 완료도를 원형 게이지로 표시
2. **상호작용 아이콘**: 각 전략마다 고유한 아이콘
3. **색상 변화**: 완료된 카드는 밝은 색상, 미완료는 회색

### 상호작용 철학
- 카드 클릭 시 확장되는 상세 정보
- 입력 필드에 타이핑할 때 실시간 피드백
- 완료 버튼 클릭 시 축하 애니메이션
- 드래그 앤 드롭으로 전략 순서 재정렬 가능

### 애니메이션
- 카드 호버 시 3D 회전 효과 (약간의 perspective)
- 완료 시 별 모양 파티클 효과
- 진행 바 업데이트 시 부드러운 숫자 카운팅 애니메이션
- 페이지 전환 시 슬라이드 인 효과

### 타이포그래피 시스템
- **제목**: Poppins Bold, 2rem, 자간 0px
- **부제목**: Poppins SemiBold, 1.3rem
- **본문**: Open Sans Regular, 1rem, 줄높이 1.6
- **강조**: Poppins Bold, 1rem

## 디자인 3: "시각적 지식 맵" (Visual Knowledge Map)

### 설계 철학
11가지 전략을 **상호 연결된 네트워크**로 표현하는 디자인. 각 전략이 독립적이면서도 전체 학습 생태계의 일부임을 시각화합니다.

### 핵심 원칙
- **네트워크 시각화**: 전략들 간의 관계를 그래프로 표현
- **공간 활용**: 중앙 중심점에서 11개 전략이 방사형으로 배치
- **계층적 정보**: 확대/축소를 통해 다양한 수준의 상세도 제공
- **색상 코딩**: 각 전략마다 고유한 색상으로 빠른 인식

### 색상 철학
- **배경**: 어두운 청회색(#0F1419) - 우주 같은 신비로움
- **노드 색상**: 11가지 전략마다 다른 색상 (무지개 스펙트럼)
  - 전략 1: 빨강(#FF4757)
  - 전략 2: 주황(#FFA502)
  - 전략 3: 노랑(#FFD93D)
  - ... (계속)
- **연결선**: 반투명 흰색(#FFFFFF40) - 전략 간 관계
- **감정적 의도**: 복잡성, 상호연결, 우주적 규모

### 레이아웃 패러다임
- **중앙 집중식 네트워크**: 학습의 중심에서 11개 전략이 방사형으로 배치
- **인터랙티브 그래프**: 마우스 호버 시 관련 전략들이 하이라이트
- **계층 구조**: 클릭 시 해당 전략이 중심으로 이동하며 상세 정보 표시
- **3D 효과**: WebGL을 활용한 깊이감 있는 시각화

### 서명 요소
1. **노드 애니메이션**: 각 노드가 부드럽게 맥박치는 효과
2. **연결선 강조**: 호버 시 관련 연결선이 밝아짐
3. **정보 툴팁**: 노드 근처에 전략명과 간단한 설명

### 상호작용 철학
- 노드 클릭 시 해당 전략이 중심으로 이동
- 드래그로 전체 그래프 회전 가능
- 마우스 휠로 확대/축소
- 더블 클릭으로 상세 정보 패널 열기

### 애니메이션
- 페이지 로드 시 노드들이 중심에서 방사형으로 퍼져나가기
- 노드 호버 시 크기 증가 + 색상 밝아지기
- 연결선 애니메이션: 경로 따라 빛이 흐르는 효과
- 노드 클릭 시 중심으로 부드럽게 이동

### 타이포그래피 시스템

- **설명**: IBM Plex Sans Regular, 0.95rem, 줄높이 1.5

## 선택된 디자인: "인터랙티브 워크숍" (Interactive Workshop)

사용자가 **직접 참여하고 실험**할 수 있는 환경을 제공하는 "인터랙티브 워크숍" 디자인을 선택했습니다. 이 접근 방식은 다음과 같은 이유로 가장 적합합니다:

1. **학습의 본질 반영**: 11가지 전략을 단순히 읽는 것이 아니라 **직접 실천**하는 경험 제공
2. **명확한 진행 추적**: 사용자가 자신의 학습 여정을 시각적으로 추적 가능
3. **즉각적 피드백**: 각 행동에 대한 시각적 반응으로 참여도 증가
4. **확장 가능성**: 향후 커뮤니티 기능이나 고급 분석 추가 가능

- 각 카드에 원형 게이지 (완료도 시각화)
- 확장 가능한 상세 정보 패널

**애니메이션:**

- 완료 시 별 모양 파티클 효과
